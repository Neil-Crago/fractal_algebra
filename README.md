## Additive Laws
- Associativity: a + (b + c) = (a + b) + c
- Commutativity: a + b = b + a
- Identity: a + 0 = a
- Inverse: a + (-a) = 0

## Multiplicative Laws
- Associativity: a \cdot (b \cdot c) = (a \cdot b) \cdot c
- Identity: a \cdot 1 = a
- Distributivity: a \cdot (b + c) = a \cdot b + a \cdot c
